# sistema-de-caixa
um sistema de gerenciamento de vendas. O que ele faz? primeiro o usuário cadastra a quantidade de produtos que deseja junto com o preço, em seguida ele tem um menu com 5 opções: listar produtos, registrar vendas, mostrar o total vendido, mostrar o produto mais caro e remover produto, ainda não fiz uma interface com HTML e CSS, mas pretendo.
